# Pulse Music v4

    Projeto desenvolvido como Trabalho de Conclusão de Curso ao Ensino Médio Técnico Integrado (Informática) no Instituto Federal do Paraná - Campus Jacarezinho.


# Qual a proposta?

    A proposta consiste no desenvolvimento de um plyer de música, baseado na linguagem Python para back-end e utilizando-se de um framework do Python (FLET) para criar a interface gráfica.
